# AGE-319 Review Evidence

## Summary

AGE-319 adds a repeatable fixture-backed pitcher start target build:

```bash
uv run python -m mlb_props_lab targets build --issue AGE-319 --run-id target-sample
uv run python -m mlb_props_lab report features --issue AGE-319 --run-id review
```

The build emits pitcher-start target rows with pre-game identity/schedule fields,
start metadata, completion metadata, and post-game strikeout labels in separate
field groups. Rows with missing target outcomes, duplicate pitcher starts,
unresolved pitcher identities, or invalid pre-game timestamps are rejected and
reported.
