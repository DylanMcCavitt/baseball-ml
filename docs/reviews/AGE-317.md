# AGE-317 Review Evidence

## Summary

AGE-317 adds a repeatable fixture-backed Statcast feature build:

```bash
uv run python -m mlb_props_lab statcast build-features --issue AGE-317 --run-id statcast-sample
uv run python -m mlb_props_lab report features --issue AGE-317
```

The build materializes registered v1 Statcast feature IDs for pitcher strikeout
skill, arsenal/stuff, pitch mix, and pitcher platoon splits. It does not emit
unregistered model features. `projected_lineup_handedness_mix` is reported as a
source gap because the Statcast pitch fixture has no projected lineup snapshot.

## Limitations

- This is a small deterministic fixture/sample build, not live Statcast ingest.
- Timestamp validity is enforced with `available_at < cutoff_at`, falling back to
  `game_date < target_game_date` when pitch availability timestamps are absent.
- Coverage gaps are explicit in `statcast_feature_coverage.csv`; missing lineup
  snapshot coverage is not filled from future or target-game rows.
